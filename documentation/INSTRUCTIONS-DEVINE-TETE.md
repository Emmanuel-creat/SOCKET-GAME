1. Copier `client/games/games.json` et le dossier `client/games/devine-tete/` dans le repo (écraser).
2. Supprimer l'ancien dossier placeholder : `git rm -r client/games/placeholder01`
3. `git add -A && git commit -m "Ajout du jeu Le Dossier (devine-tete)" && git push` → Render redéploie.

⚠️ Les assets pèsent ~107 Mo (135 photos) : le premier push sera long, c'est normal.
Les URLs des images sont anonymisées (sa/sb/sc…) pour que l'inspecteur réseau ne révèle pas la réponse.
